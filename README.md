timeline
======

> Author : Wang Jie <i@i8e.net>

### Introduce

It's a simple multi-user timeline application.

### Usage

0. git clone this repo
0. run `npm install`
0. start mongodb on port 27017
0. run `node app`

### Used modules

 - express
 - handlebars
 - mongoose
 - moment
 - uuid
 - intercooler
